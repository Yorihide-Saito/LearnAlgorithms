### B問題: ビンゴカウント

**使用アルゴリズム**: #実装 #全探索

```cpp
#include<iostream>
#include<vector>
using namespace std;

int main() {
    int H, W, N;
    cin >> H >> W >> N;
    vector<vector<int>> grid(H, vector<int>(W));
    for (int i = 0; i < H; i++) {
        for (int j = 0; j < W; j++) {
            cin >> grid[i][j];
        }
    }

    vector<int> ans(H, 0);
    while(N--) {
        int B;
        cin >> B;
        for (int i = 0; i < H; i++) {
            for (int j = 0; j < W; j++) {
                if (grid[i][j] == B) ans[i]++;
            }
        }
    }

    int max_ans = 0;
    for (int a : ans) {
        max_ans = max(a, max_ans);
    }
    cout << max_ans << endl;
    return 0;
}
```

**概要**: H×Wのグリッドで、N個の数字が呼ばれたとき、各行で何個マッチするかを数え、最大値を出力

### C問題: トナカイの配置

**使用アルゴリズム**: #ソート #累積和 #二分探索 #貪欲法

```cpp
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;
using ll = long long;

int main() {
    int T;
    cin >> T;
    while(T--) {
        int N;
        cin >> N;
        vector<pair<ll, ll>> reindeer(N);
        for (int i = 0; i < N; i++) {
            cin >> reindeer[i].first >> reindeer[i].second;
        }

        // W + P でソート
        sort(reindeer.begin(), reindeer.end(), [](const auto& a, const auto& b) {
            return a.first + a.second < b.first + b.second;
        });

        // 累積和を構築
        vector<ll> sumW(N + 1, 0), sumP(N + 1, 0);
        for (int i = 0; i < N; i++) {
            sumW[i + 1] = sumW[i] + reindeer[i].first;
        }
        for (int i = N - 1; i >= 0; i--) {
            sumP[i] = sumP[i + 1] + reindeer[i].second;
        }

        // 二分探索（めぐる式）
        int l = 0, r = N;
        while(r - l > 1) {
            int m = (l + r) / 2;
            if (sumW[m] <= sumP[m]) l = m;
            else r = m;
        }

        cout << l << endl;
    }
    return 0;
}
```

**概要**: トナカイをW+Pの順にソートし、累積和を使って条件を満たす最大個数を二分探索で求める

**使用テクニック**:

- カスタムラムダソート
- 前方向/後方向の累積和
- めぐる式二分探索

### D問題: 差の総和（MOD計算）

**使用アルゴリズム**: #ソート #累積和 #lower_bound #MOD #二分探索

```cpp
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;
using ll = long long;

int main() {
    ll N, M;
    cin >> N >> M;
    vector<ll> A(N), B(M);
    for (int i = 0; i < N; i++) cin >> A[i];
    for (int i = 0; i < M; i++) cin >> B[i];

    sort(A.begin(), A.end());
    sort(B.begin(), B.end());

    // Aの累積和
    vector<ll> sumA(N + 1, 0);
    for (int i = 0; i < N; i++) {
        sumA[i + 1] = sumA[i] + A[i];
    }

    ll ans = 0;
    const ll MOD = 998244353;
    for (int m = 0; m < M; m++) {
        ll x = B[m];
        ll idx = lower_bound(A.begin(), A.end(), x) - A.begin();

        // x未満の要素との差の和
        ll left = x * idx - sumA[idx];
        // x以上の要素との差の和
        ll right = (sumA[N] - sumA[idx]) - x * (N - idx);

        ans = (ans + left + right) % MOD;
    }

    cout << ans << endl;
    return 0;
}
```

**概要**: 2つの配列A, Bについて、すべてのペア(A[i], B[j])の差の絶対値の総和をMODで求める

**使用テクニック**:

- ソート
- 累積和
- lower_bound
- イテレータ減算でインデックス取得
- MOD計算

## Tips まとめ

### 1. カスタムラムダソート

```cpp
sort(vec.begin(), vec.end(), [](const auto& a, const auto& b) {
    return a.first + a.second < b.first + b.second;
});
```

- pairやカスタム条件でソートする際に使用
- `auto&` で型推論を活用

### 2. 累積和（Prefix Sum）

```cpp
vector<ll> sum(N + 1);
for (int i = 0; i < N; i++) {
    sum[i + 1] = sum[i] + A[i];
}
```

- 区間和を O(1) で計算可能
- sum[i] は A[0]～A[i-1] の和

### 3. 後方向累積和

```cpp
for (int i = N - 1; i >= 0; i--) {
    sumP[i] = sumP[i + 1] + P[i];
}
```

- 右から左への累積和
- sumP[i] は P[i]～P[N-1] の和

### 4. めぐる式二分探索

```cpp
int l = 0, r = N;  // [l, r) の半開区間
while(r - l > 1) {
    int m = (l + r) / 2;
    if (check(m)) l = m;  // 条件を満たす
    else r = m;           // 条件を満たさない
}
// 答えは l
```

- 境界値を探す際の定石
- ループ不変条件: check(l) = true, check(r) = false

### 5. lower_bound でインデックス取得

```cpp
ll idx = lower_bound(A.begin(), A.end(), x) - A.begin();
```

- `x` 以上の最初の要素の位置を取得
- イテレータから begin() を引くことでインデックス化

### 6. MOD計算の注意

```cpp
const ll MOD = 998244353;
ans = (ans + value) % MOD;
```

- 足し算/掛け算の度に MOD を取る
- 負の数になる可能性がある場合は `(value % MOD + MOD) % MOD`

### 7. 範囲for文

```cpp
for (auto a : ans) {
    max_ans = max(a, max_ans);
}
```

- コンテナの全要素を走査
- `auto&` で参照にすると値のコピーを避けられる

## 解法パターン

### パターン1: ソート + 累積和 + 二分探索

C問題やD問題で使用。

1. データをソート
2. 累積和を構築
3. 二分探索や線形探索で答えを求める

### パターン2: 差の絶対値の総和

D問題のように、ソートしてから累積和を使うことで効率的に計算可能。

- x未満の要素: `x * count - sum`
- x以上の要素: `sum - x * count`
