## 概要

いもす法。**区間加算を複数回行い、各点での値を求める**テクニック。差分配列 + 累積和の応用。

## 計算量

- 区間加算: `O(1)`
- 復元: `O(N)`
- 全体: `O(N + Q)` (Q: 区間加算の回数)

## 実装コード

### 1次元いもす法

```cpp
#include<iostream>
#include<vector>
using namespace std;

int main() {
    int N, Q;
    cin >> N >> Q;

    vector<int> imos(N + 1, 0);

    // Q個の区間加算
    for (int i = 0; i < Q; i++) {
        int L, R, X;
        cin >> L >> R >> X;  // 区間 [L, R) に +X

        imos[L] += X;
        imos[R] -= X;
    }

    // 累積和で復元
    vector<int> A(N);
    A[0] = imos[0];
    for (int i = 1; i < N; i++) {
        A[i] = A[i - 1] + imos[i];
    }

    // 結果を出力
    for (int i = 0; i < N; i++) {
        cout << A[i] << " ";
    }
    cout << endl;

    return 0;
}
```

### 2次元いもす法

```cpp
// H×W のグリッドに対する2次元いもす法
vector<vector<int>> imos(H + 1, vector<int>(W + 1, 0));

// 長方形 [y1, y2) × [x1, x2) に +val を加算
void add_rect(int y1, int x1, int y2, int x2, int val) {
    imos[y1][x1] += val;
    imos[y1][x2] -= val;
    imos[y2][x1] -= val;
    imos[y2][x2] += val;
}

// 累積和で復元
vector<vector<int>> A(H, vector<int>(W));

// 横方向に累積和
for (int i = 0; i < H; i++) {
    for (int j = 0; j < W; j++) {
        if (j > 0) imos[i][j] += imos[i][j - 1];
    }
}

// 縦方向に累積和
for (int i = 0; i < H; i++) {
    for (int j = 0; j < W; j++) {
        if (i > 0) imos[i][j] += imos[i - 1][j];
        A[i][j] = imos[i][j];
    }
}
```

## 使用例

- カレンダー問題（予定の重複数を数える）
- グリッド上の長方形領域への一括加算
- イベントソートの代替手法

## よくある問題パターン

```cpp
// 例: N日間のカレンダーで、Q個の予定がある。
// 各予定は [L, R) の期間に存在する。
// 各日に何個の予定があるか求めよ。

int N, Q;
cin >> N >> Q;

vector<int> cnt(N + 1, 0);

for (int i = 0; i < Q; i++) {
    int L, R;
    cin >> L >> R;

    cnt[L]++;
    cnt[R]--;
}

// 復元
for (int i = 1; i < N; i++) {
    cnt[i] += cnt[i - 1];
}

// cnt[i] = i日目の予定数
for (int i = 0; i < N; i++) {
    cout << cnt[i] << endl;
}
```

## 注意点

- 配列のサイズは `N + 1` にする（R の位置に -X を書き込むため）
- 半開区間 `[L, R)` で考えると実装がシンプル
- 2次元の場合、縦横の順番に注意
