## 概要

累積和（Prefix Sum）。配列の**区間和をO(1)で求める**テクニック。前処理 O(N)、クエリ O(1)。

## 計算量

- 前処理: `O(N)`
- クエリ: `O(1)`

## 実装コード

### 1次元累積和

```cpp
#include<iostream>
#include<vector>
using namespace std;

int main() {
    int N;
    cin >> N;
    vector<int> A(N);
    for (int i = 0; i < N; i++) cin >> A[i];

    // 累積和を作る（S[i] = A[0] + A[1] + ... + A[i-1]）
    vector<long long> S(N + 1, 0);
    for (int i = 0; i < N; i++) {
        S[i + 1] = S[i] + A[i];
    }

    // 区間 [L, R) の和を求める
    int L, R;
    cin >> L >> R;
    long long sum = S[R] - S[L];
    cout << sum << endl;

    return 0;
}
```

### 2次元累積和

```cpp
// H×W のグリッドの累積和
vector<vector<int>> A(H, vector<int>(W));
vector<vector<long long>> S(H + 1, vector<long long>(W + 1, 0));

// 累積和を作る
for (int i = 0; i < H; i++) {
    for (int j = 0; j < W; j++) {
        S[i + 1][j + 1] = S[i][j + 1] + S[i + 1][j] - S[i][j] + A[i][j];
    }
}

// 長方形 (y1, x1) から (y2, x2) の和を求める（y2, x2は含まない）
long long rect_sum(int y1, int x1, int y2, int x2) {
    return S[y2][x2] - S[y1][x2] - S[y2][x1] + S[y1][x1];
}
```

## 使い方

```cpp
// 1次元の例
vector<int> A = {3, 1, 4, 1, 5};
vector<long long> S(6);
S[0] = 0;
for (int i = 0; i < 5; i++) {
    S[i + 1] = S[i] + A[i];
}

// 区間 [1, 4) の和 = A[1] + A[2] + A[3]
cout << S[4] - S[1] << endl;  // 1 + 4 + 1 = 6
```

## 使用例

- 区間和クエリ
- 2次元グリッドの長方形領域の和
- 差分を使った更新との組み合わせ

## 応用

**差分配列**: 累積和の逆操作。区間加算を O(1) で行い、最後に累積和で復元

```cpp
// 区間 [L, R) に +x を加える操作を高速化
vector<int> diff(N + 1, 0);

// 区間 [L, R) に +x
diff[L] += x;
diff[R] -= x;

// 最後に累積和で復元
vector<int> A(N);
A[0] = diff[0];
for (int i = 1; i < N; i++) {
    A[i] = A[i - 1] + diff[i];
}
```
