## 概要

ダイクストラ法（Dijkstra's Algorithm）。**重み付きグラフの単一始点最短経路**を求めるアルゴリズム。負の辺がない場合に使用。

## 計算量

- 時間: `O((V + E) log V)` (優先度付きキュー使用時)
- 空間: `O(V)`

## 実装コード

```cpp
#include<iostream>
#include<vector>
#include<queue>
using namespace std;
using ll = long long;

const ll INF = 1e18;

// グラフ: G[v] = {(隣接頂点, コスト), ...}
vector<vector<pair<int, ll>>> G;

vector<ll> dijkstra(int start, int N) {
    vector<ll> dist(N, INF);
    priority_queue<pair<ll, int>, vector<pair<ll, int>>, greater<>> pq;

    dist[start] = 0;
    pq.push({0, start});

    while (!pq.empty()) {
        auto [d, v] = pq.top(); pq.pop();

        // 既により短い経路が見つかっている場合はスキップ
        if (dist[v] < d) continue;

        for (auto [nv, cost] : G[v]) {
            ll nd = dist[v] + cost;
            if (dist[nv] <= nd) continue;

            dist[nv] = nd;
            pq.push({nd, nv});
        }
    }

    return dist;
}

int main() {
    int N, M;
    cin >> N >> M;

    G.resize(N);

    for (int i = 0; i < M; i++) {
        int a, b;
        ll c;
        cin >> a >> b >> c;
        a--; b--;  // 0-indexed

        G[a].push_back({b, c});
        // 無向グラフの場合
        // G[b].push_back({a, c});
    }

    vector<ll> dist = dijkstra(0, N);

    for (int i = 0; i < N; i++) {
        if (dist[i] == INF) {
            cout << "INF" << endl;
        } else {
            cout << dist[i] << endl;
        }
    }

    return 0;
}
```

## 使用例

- 最短経路問題
- 移動コストの最小化
- ネットワークルーティング

## 応用

**経路復元**: 最短経路そのものを求める

```cpp
vector<ll> dist(N, INF);
vector<int> prev(N, -1);  // 前の頂点を記録
priority_queue<pair<ll, int>, vector<pair<ll, int>>, greater<>> pq;

dist[start] = 0;
pq.push({0, start});

while (!pq.empty()) {
    auto [d, v] = pq.top(); pq.pop();
    if (dist[v] < d) continue;

    for (auto [nv, cost] : G[v]) {
        ll nd = dist[v] + cost;
        if (dist[nv] <= nd) continue;

        dist[nv] = nd;
        prev[nv] = v;  // 経路を記録
        pq.push({nd, nv});
    }
}

// 経路復元（goalからstartへ逆順）
vector<int> path;
int cur = goal;
while (cur != -1) {
    path.push_back(cur);
    cur = prev[cur];
}
reverse(path.begin(), path.end());
```

## 注意点

- 負の辺がある場合はベルマンフォード法を使う
- INF は問題に応じて適切に設定（オーバーフロー注意）
- `dist[v] < d` のチェックで枝刈り
